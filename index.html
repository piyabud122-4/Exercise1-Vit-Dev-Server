What is the role of the index.html file?

= It is the default or main HTML file that a web server loads when a user visits a website or a folder on a website.

What is the URL you used to view the page? Is it HTTP or HTTPS?

= The URL that I used to view the page is http://localhost:63342/, so it is HTTP.

What HTTP status code do you see for index.html? What does it mean?

= I see the 200 HTTP status code, which means OK and indicates that the request for the resource was successful.

How can you tell the page is being served locally vs from a real domain?

= You can check by looking at the URL in the browser's address bar. If it looks like http://localhost:63342/, it's a local page.

But if it looks like https://www.google.com/, it's a real domain.
